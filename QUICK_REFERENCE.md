# 🚀 Quick Reference Guide

## Toolbar Buttons

| Button | Icon | Function | Shortcut |
|--------|------|----------|----------|
| Run | ▶ | Execute your code | Enter (in some modes) |
| New | 📄 | Load language template | - |
| Share | 🔗 | Generate shareable link | - |
| Download | 💾 | Save code to file | - |
| Upload | 📁 | Import code from file | - |
| Snippets | 📚 | Open snippets library | - |
| Analyze | 📊 | Show code statistics | - |
| History | 🕒 | View execution history | - |
| Theme | 🌙 | Toggle dark/light mode | - |

---

## Common Workflows

### 1. Quick Code Execution
```
1. Select language
2. Write/paste code
3. Click ▶ Run
4. View output
```

### 2. Share Code with Others
```
1. Write your code
2. Click 🔗 Share
3. Link copied to clipboard
4. Share the link
```

### 3. Save Code Snippet
```
1. Write code
2. Click 📚 Snippets
3. Click 💾 Save Current
4. Enter name
5. Done!
```

### 4. Upload Existing File
```
1. Click 📁 Upload
2. Select file
3. Code loads automatically
```

### 5. Analyze Your Code
```
1. Write code
2. Click 📊 Analyze
3. View statistics popup
```

## Input/Output

### Providing Input
1. Click in "Input (stdin)" area
2. Type your input
3. Each line = one input value
4. Click ▶ Run

### Saving Output
1. Run your code
2. Click **💾 Save Output** (in toolbar)
3. File downloads automatically

## Code Analysis Metrics

When you click **📊 Analyze**, you get:

- **Total Lines** - All lines including empty
- **Non-empty Lines** - Lines with code
- **Characters** - Total character count
- **Words** - Total word count
- **Functions** - Detected functions (language-specific)
- **Classes** - Detected classes (language-specific)
- **Imports** - Import statements (language-specific)

### Code Won't Run
- ✅ Check language selection
- ✅ Verify syntax
- ✅ Check for errors in output

### Upload Failed
- ✅ Check file size (max 16MB)
- ✅ Verify file format
- ✅ Try different browser

### Share Link Not Working
- ✅ Check internet connection
- ✅ Verify link copied correctly
- ✅ Try generating new link

### Output Not Showing
- ✅ Check if code has print statements
- ✅ Look for errors in output panel
- ✅ Verify input is correct

## Language-Specific Notes

### Python
- Use `print()` for output
- Indentation matters
- No semicolons needed

### Java
- Must have `public static void main`
- Class name should be `Main`
- Semicolons required

### JavaScript
- Use `console.log()` for output
- Runs in Node.js environment
- Semicolons optional

### C/C++
- Must have `main()` function
- Use `printf()` or `cout`
- Semicolons required

### Go
- Must have `package main`
- Must have `func main()`
- Use `fmt.Println()`
